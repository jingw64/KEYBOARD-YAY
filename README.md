# J&R's Keyboard Project

## Overview

For this project, we wish to learn the process of making a keyboard. Due to how well documented and relatively simple keyboards are, we decided that creating one from the ground up will be a good first project for developing important skills.

After the completion of this project, we will:

- Understand the basics of navigating git and github
- Know the process of designing a PCB
- Develop collaboration skills
- Gain experience in writing software for an embedded system
